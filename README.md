# Power BI Sales Dashboard

This project presents an end-to-end **Power BI Sales Performance Dashboard** designed to analyze sales trends over a 4-year period.

---

## Key Features

- **Data Modeling (Star Schema)**  
  Built a structured data model linking Fact and Dimension tables to optimize performance and support efficient analysis.

- **DAX Calculations**  
  Created custom DAX measures to compute key business metrics, including:  
  - Total Amount  
  - Total Due  
  - Total Freight  
  - Total Tax  
  - Monthly & Year-to-Date trends  

- **DAX Date Table**  
  Implemented a Date table using DAX to enable time-intelligence functions and dynamic filtering (Year, Quarter, Month).

- **Interactive Visualizations & KPIs**  
  Designed clear and interactive visuals including:  
  - KPI Cards  
  - Line Chart (Monthly Orders)  
  - Treemap (Orders by Category)  
  - Donut Chart (Orders by Customer Name)  

---

## Tools & Technologies

- Power BI Desktop  
- DAX  
- Data Modeling (Star Schema)  
- Power Query  

---

## Key Insights Uncovered

- Year-over-year sales performance and trends  
- Top-performing product categories and customers  
- Revenue distribution across multiple metrics (Amount, Due, Freight, Tax)
